Proyectos

Base de Datos SkyBase

Sistema de gestión de empleados con interfaz gráfica

Aplicación de escritorio desarrollada en Python con Tkinter para gestionar datos de empleados. Permite registrar nombre, cargo y salario, con funcionalidades de guardado automático en JSON, exportación a PDF y CSV, cambio de idiomas (español/inglés) y temas claro/oscuro. Incluye sistema de menús completo y validación de datos.

Características principales:
• Interfaz gráfica intuitiva con Tkinter
• Persistencia de datos en formato JSON
• Exportación a PDF y CSV
• Sistema multiidioma (ES/EN)
• Temas personalizables
• Validación de entrada de datos

Tecnologías: Python, Tkinter, JSON, ReportLab, CSV

Visualización de Terremotos en Tiempo Real

Sistema de monitoreo y visualización de actividad sísmica global

Aplicación desarrollada en Python que consume la API de USGS para obtener datos de terremotos de las últimas 24 horas. Procesa la información sísmica y genera visualizaciones interactivas en un mapa mundial, mostrando la magnitud, ubicación y profundidad de cada evento. Los datos se exportan automáticamente a CSV para análisis posterior.

Características principales:
• Consumo de API REST de USGS en tiempo real
• Procesamiento de datos geoespaciales con Pandas
• Visualización interactiva con mapas globales
• Filtrado por magnitud sísmica
• Exportación automática a formato CSV
• Información detallada por evento sísmico

Tecnologías: Python, Requests, Pandas, Plotly, USGS API, GeoJSON

Análisis de Repositorios GitHub

Dashboard de análisis de popularidad en repositorios de GitHub

Herramienta de análisis que utiliza la API de GitHub para identificar y visualizar los repositorios más populares de la plataforma. Genera gráficos de barras interactivos mostrando estadísticas de estrellas, con información detallada de cada repositorio incluyendo descripciones y métricas de popularidad. Ideal para análisis de tendencias en desarrollo de software.

Características principales:
• Integración con GitHub API REST
• Análisis de métricas de popularidad
• Visualización interactiva con gráficos
• Ranking automático por estrellas
• Información detallada de repositorios
• Interfaz moderna con tema oscuro

Tecnologías: Python, Requests, Plotly, GitHub API, JSON
